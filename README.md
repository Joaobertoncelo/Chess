# JogoXadrezEmJava
Jogo desenvolvido para a disciplina de "Programação orientada a objetos", com o objetivo final de ter um jogo de xadrez funcional com interface em Java.

O projeto foi refatorado 2 anos depois, utilizando ferramentas de refatoração como CODEMR e boas práticas baseadas nos livros "Clean Code" e "Refatoração".
